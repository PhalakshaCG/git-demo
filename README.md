# Git Hands-On Demo: Express App

## Prerequisites
- Git installed on your system
- Node.js and npm installed
- Basic understanding of Git commands

---

## Step 1: Clone the Repository
1. Open your terminal.
2. Clone this repository:
   ```bash
   git clone https://github.com/PhalakshaCG/demo-app.git
   ```
3. Navigate into the project directory:
   ```bash
   cd demo-app
   ```

---

## Step 2: Checkout to a Custom Branch
1. Create and switch to a new branch:
   ```bash
   git checkout -b ${your name here}
   ```
2. Verify that you are on the new branch:
   ```bash
   git branch
   ```
   The branch you are on will be highlighted with an asterisk (`*`).
